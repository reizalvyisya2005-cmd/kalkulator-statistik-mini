Tentang Aplikasi

Statistik Pro App adalah aplikasi statistik berbasis HTML, CSS, dan JavaScript yang dibuat untuk membantu mahasiswa dalam mengolah data penelitian, memahami hasil analisis, serta mempelajari konsep statistik dasar hingga pengujian hipotesis.

Tujuan Aplikasi

  • Membantu pengolahan data penelitian secara cepat.
  • Menyediakan hasil statistik yang rapi dan mudah dibaca.
  • Memberikan interpretasi otomatis agar hasil lebih mudah dipahami.
  • Menjadi media belajar statistik penelitian bagi pemula.

Materi & Panduan Statistik

Bagian ini berisi ringkasan teori dan panduan sederhana dari materi statistik yang sering dipakai dalam penelitian.

1. Uji Deskriptif

Tujuan: menggambarkan data secara umum tanpa menarik kesimpulan inferensial.

  • Menampilkan jumlah data, mean, median, modus, minimum, maksimum, range, varians, dan standar deviasi.
  • Cocok untuk melihat kecenderungan pusat dan sebaran data.

Mean: x̄ = Σx / n

Range: nilai maksimum − nilai minimum

Standar deviasi: ukuran penyebaran data dari rata-rata

2. Uji Validitas

Tujuan: menguji apakah setiap item pertanyaan benar-benar mengukur variabel yang ingin diukur.

  • Menggunakan korelasi item dengan total skor.
  • Item dinyatakan valid jika r hitung ≥ r tabel.

Dasar keputusan:

  • Jika r hitung ≥ r tabel → item valid
  • Jika r hitung < r tabel → item tidak valid

3. Uji Reliabilitas (Cronbach’s Alpha)

Tujuan: menguji konsistensi jawaban antar item dalam satu variabel.

  • Semakin tinggi nilai alpha, semakin konsisten instrumen.
  • Umumnya reliabel jika α ≥ 0,70.

Cronbach’s Alpha:

α = (k / (k − 1)) × (1 − (Σ varians item / varians total))

4. Kalkulator Nilai Z

Tujuan: menentukan nilai Z berdasarkan tingkat kepercayaan atau digunakan pada uji hipotesis proporsi/rata-rata tertentu.

  • 90% → Z = 1,645
  • 95% → Z = 1,96
  • 99% → Z = 2,576

5. Kalkulator Ukuran Sampel

Tujuan: menentukan jumlah sampel minimal penelitian.

Populasi besar / tidak diketahui:

n = (Z² × p × (1 − p)) / e²

Jika populasi terbatas:

nf = n / (1 + ((n − 1) / N))

  • Z = tingkat kepercayaan
  • p = proporsi awal (biasanya 0,5 jika belum diketahui)
  • e = margin of error
  • N = jumlah populasi

6. Uji Hipotesis

Tujuan: menguji dugaan penelitian berdasarkan data sampel.

  • Hipotesis nol (H₀): tidak ada pengaruh / tidak ada perbedaan / tidak ada hubungan.
  • Hipotesis alternatif (H₁): ada pengaruh / ada perbedaan / ada hubungan.

Keputusan umum:

  • Jika p-value < 0,05 → tolak H₀
  • Jika p-value ≥ 0,05 → gagal menolak H₀

Alur Belajar yang Disarankan

  1. Pahami dulu statistik deskriptif.
  2. Lanjut ke validitas item pertanyaan.
  3. Cek reliabilitas instrumen.
  4. Pelajari ukuran sampel dan nilai Z.
  5. Terakhir, pahami logika uji hipotesis.
